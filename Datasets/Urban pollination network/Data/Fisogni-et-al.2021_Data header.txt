Dataset used in the article "Seasonal trajectories of plant-pollinator networks differ along an urbanization gradient".

This file includes plant-pollinator interactions sampled in each site (1-12) at each sampling event (6 events).
Each row is a single 'insect pollinator X plant' interaction.
The file is .txt with tab-separated values.
